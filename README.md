# دستیار صوتی + پیامک (نسخه اولیه)

## این پروژه چیه؟
یک اپ اندرویدی که با فشردن دکمه میکروفون، دستور صوتی فارسی می‌گیره و دو تا کار انجام می‌ده:
1. آخرین پیامک‌های دریافتی رو با صدا برات می‌خونه
2. با گفتن اسم یا شماره، پیامک جدید می‌فرسته

## نحوه‌ی باز کردن پروژه
1. Android Studio رو نصب کن (نسخه‌ی 2023 یا جدیدتر): https://developer.android.com/studio
2. از منوی `File > Open` پوشه‌ی `VoiceAssistant` رو انتخاب کن
3. Android Studio خودش gradle wrapper رو دانلود و sync می‌کنه (نیاز به اینترنت داره)
4. گوشیت رو با یواس‌بی وصل کن و حالت Developer Options / USB Debugging رو فعال کن
5. دکمه Run (▶) رو بزن

## دستورهای پشتیبانی‌شده (نسخه‌ی فعلی)
- «پیامک‌های جدید رو بخون» → آخرین ۳ پیامک رو با صدا می‌خونه
- «به [اسم مخاطب یا شماره] پیامک بده بنویس [متن]» → پیامک می‌فرسته

مثال: «به علی پیامک بده بنویس سلام امروز میای؟»

## محدودیت‌های فعلی (چیزهایی که باید بعداً اضافه کنیم)
- تشخیص دستور فعلاً بر پایه‌ی قوانین ساده (regex) است، نه هوش مصنوعی واقعی.
  برای دستورهای پیچیده‌تر یا فهمیدن منظور آزادانه، باید متن رو به یک مدل زبانی
  (مثل Claude API) بفرستیم تا intent و پارامترها رو دقیق دربیاره.
- در حال حاضر باید دکمه رو بزنی و صحبت کنی (push-to-talk)؛ اگر بخوای همیشه
  گوش بده (wake word مثل "هی دستیار")، نیاز به کتابخونه‌ای مثل Porcupine داریم
  که مصرف باتری و مجوزهای بیشتری می‌طلبه.
- خوندن خودکار پیامک‌های تازه‌رسیده با صدا (بدون اینکه خودت بخوای) هنوز فعال
  نشده — کدش در SmsReceiver.kt آماده‌ست، فقط باید TTS رو بهش وصل کنیم.
- این اپ باید به‌صورت دستی (APK) نصب بشه، نه از طریق Google Play، چون Play
  اپ‌هایی که دسترسی SMS می‌خوان رو محدود کرده.

## قدم بعدی
وقتی این بخش رو تست کردی و راضی بودی، می‌ریم سراغ:
- اتصال به روبیکا (برای چک کردن گروه‌ها) — این بخش پیچیده‌تره چون روبیکا
  API عمومی رسمی برای خوندن پیام‌های شخصی نداره
- یا wake-word (بدون نیاز به فشردن دکمه)
- یا وصل کردن CommandProcessor به یک مدل هوش مصنوعی برای فهم بهتر دستورها
